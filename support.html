About RedactPDF

RedactPDF is a professional privacy protection tool designed to help you automatically and manually remove sensitive information from PDF documents.

  • Automatic sensitive data detection
  • Manual redaction mode
  • Region-based rules
  • Undo / Redo support
  • Secure export

Getting Started

1. Tap Import to load a PDF file.
2. Choose Automatic or Manual mode.
3. Select applicable regions.
4. Tap Detect to scan.
5. Tap Apply to redact.
6. Tap Export to save.

Redaction Modes

Automatic Mode

Automatically detects:

  • ID numbers
  • Phone numbers
  • Email addresses
  • Bank cards

Manual Mode

Draw directly on the document to hide selected areas.

Manual mode is recommended when automatic detection misses content.

Region & Rule Management

To reduce false detection, RedactPDF allows you to enable rules based on regions.

  • China
  • USA
  • Japan
  • Korea
  • France
  • Germany
  • Spain
  • India
  • Philippines
  • Peru
  • UK
Tap Region → Select countries → Confirm.
Enabling unnecessary regions may cause false redaction.

Undo & Redo

  • Undo: Cancel last action
  • Redo: Restore canceled action
Undo/Redo only applies to manual redactions.

Frequently Asked Questions

Why can't I open a PDF?

Please make sure the file is not corrupted and you are using the latest version.

Does RedactPDF upload my files?

No. All processing is done locally.

Can I recover redacted content?

No. Once exported, content cannot be recovered.

Why is nothing detected?

Some PDFs are scanned images. Please use manual mode.

Why does manual redaction sometimes fail?

In Manual Mode, redaction works on a per-page basis. If your selection spans across multiple pages, the redaction may not be applied correctly. Please make sure to select within a single page.
